V1.1 内部审阅

产品功能调整需求

DodoWheels WMS - 产品信息展示逻辑更新

1. 核心变更说明

本次调整旨在提升客户端用户对轮胎季节属性的识别效率。我们将通过解析 Data 页签中的 FINISH 字段，自动在产品主图右上角渲染对应的特征图标。

2. 数据映射关系

后台 FINISH 字段输入 前端显示效果 图标状态
SUMMER 默认夏季标识 保持不变
WINTER 默认冬季标识 保持不变
ALL SEASON 新版四季图标 (Icon 1) 新增调整
ALL WEATHER 新版全天候图标 (Icon 2) 新增调整

3. 交互逻辑与视觉参考

数据源：后台配置页

Tab: Data
Field: [ FINISH ]
Input: "ALL WEATHER"

系统将自动抓取该输入值

呈现：客户端效果

🛞
☀️

图标将自动悬浮于图片右上角

4. 技术要求

  • 实时性： 修改保存后，前端 CDN 缓存需在合理时间内刷新展示新图标。
  • 兼容性： 图标需适配 Retina 屏幕，确保在高分辨率下不模糊。
  • UI 规范： 图标距离图片边缘间距需统一设定为 8px (Padding)。
