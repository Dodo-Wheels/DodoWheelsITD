模块 01

系统看板

实时总览大屏，汇聚所有业务模块的核心数据，管理者无需切换页面即可掌握全局运营状态。

  • 多维统计：订单量、容器数、批次数、司机 / 车辆 / 叉车状态一屏呈现
  • 快速导航：点击看板中任意模块卡片，直接跳转至对应功能页面
  • 实时刷新：数据来源于云端或本地，反映当前最新状态
实时数据快速导航多维概览
📋
模块 02

订单管理

统一处理业务订单的校验、查询与状态管理，支持云端实时同步与本地 Excel 文件两种数据源。

  • 订单校验器：逐条核验订单合法性，异常订单即时标红提示，支持批量导入
  • 订单查询：只读模式快速检索，按单号、状态、司机、车辆、时间多维筛选
  • 订单撤销：专属撤单视图，撤单操作有独立权限控制，避免误操作
  • 自动保存：编辑后通过 File System Access API 实时写回本地 Excel 文件
  • 全局搜索：顶栏 300ms 防抖搜索，快速定位任意单号
订单校验只读查询 撤单处理批量导入云端/本地
🚢
模块 03

集装箱管理

管理每一个集装箱从海外发出到入库完成的完整生命周期，支持多仓库并行操作。

海运中 已到港 卸货中 已完成 已退柜
  • 多仓库支持：MT、GT、QC 仓同时管理，按仓筛选查看
  • ETA 区间筛选：可按预计到港日期范围过滤柜次，精准规划卸货计划
  • 货品明细：每个集装箱下挂载 SKU / 品牌 / 描述 / 实收数量，支持逐项核对
  • 实收数量同步：实收与预收对比，差异自动高亮，云端即时更新
  • Excel 导出：一键导出柜次明细至 Excel，便于对账归档
5种状态流转多仓并行 ETA筛选实收核对
📡
模块 04

到货预报

基于集装箱数据自动生成到货预报视图，让仓库提前安排人力与卸货位，减少压港等待。

  • 预报汇总：按 ETA 日期聚合展示即将到港的柜次及货品清单
  • 库存联动：自动关联在途货品的 SKU，与现有库存进行对比预测
  • 云端实时：数据直接来源于集装箱模块，无需重复录入
预报汇总SKU联动仓库预排
📥
模块 05

入库单创建

为每次实际入库行为创建标准化入库单，涵盖四种入库类型，覆盖所有入库来源场景。

待收货 收货中 已完成 已上架
  • 四种入库类型：集装箱入库 / 采购入库 / 调拨入库 / 退货入库，场景全覆盖
  • 货品行管理：每单下可添加多条 SKU 明细，含预计数量与实际收货数量
  • 关联集装箱：集装箱入库类型可直接选择已有柜次，数据自动带入
  • 状态流转：待收货 → 收货中 → 已完成 → 已上架，全程可追溯
  • 打印功能：一键打印入库单，现场收货凭证可纸质留存
4种入库类型多SKU明细 柜次关联打印支持
📌
模块 06

上架管理

承接已完成收货的入库单，将货品精准分配至仓库的具体货位，完成最后一公里入库动作。

  • 货位分配：从已有货位库中选择目标位置，为每件商品指定存放货架
  • 入库单联动：自动拉取"已完成"状态的入库单及其 SKU 明细
  • 上架记录：每次上架操作均生成带时间戳的记录，可追溯历史
  • 多仓位支持：支持同一 SKU 分散存放在多个货位，按位置分别记录数量
货位指定入库单联动操作记录
📤
模块 07

出库单创建

创建标准出库订单，指定司机、车辆、客户及出库货品明细，系统自动生成唯一出库单号。

待拣货 已拣货 出库完成
  • 自动编号：按日期自动生成 OB+日期+序号格式的唯一出库单号
  • 关键要素：出库日期、时段、客户名、仓库、司机、车辆、备注一单录入
  • 多货品行：每单可添加多条 SKU 行（SKU / 商品名 / 数量），动态增减
  • 状态驱动：待拣货 → 已拣货 → 出库完成，与下架环节联动
  • 打印 & 撤回：支持打印出库单，已完成单据可撤回重新处理
自动单号司机/车辆 多SKU行打印撤回
📦
模块 08

下架管理

承接出库单，仓库员工按单从货架上取货，完成实物下架操作并更新订单状态。

  • 待拣货列表：自动聚合当日所有待拣货出库单，按订单分组展示
  • 逐单操作：点击确认拣货，状态即时更新为"已拣货"
  • 完成确认：已拣货后再次确认完成，订单进入"出库完成"终态
  • 打印拣货单：支持打印拣货单，仓库现场操作的纸质凭证
待拣聚合逐单确认打印拣货单
🗺️
模块 09

订单派单分配

将已接受的订单按区域、仓库批量分配给指定司机和车辆，生成每日配送计划清单。

  • 四维筛选：按方向（东/南/西/北）、仓库、司机、车辆组合过滤订单
  • 批量指派：勾选多条订单，一次性分配给同一司机/车辆
  • 派单查询：查询视图支持按司机/车辆查看历史派单记录
  • 汇总报表：自动汇总每次派单的总件数、分区占比，便于调度分析
  • 明细视图：逐条展示每笔订单的出库详情，支持 Excel 导出
4个维度筛选批量分配 汇总报表Excel导出

模块 10

客户自提记录

记录客户自行来仓提货的操作，支持确认提货及查询历史自提记录。

  • 自提确认：客户到仓后，操作员按单号确认提货，状态自动更新
  • 提货记录：保留完整的自提时间、单号、客户信息，便于纠纷追溯
  • 日期筛选：按提货日期快速查询历史记录
提货确认历史记录日期筛选

模块 11

批次管理

管理出库批次数据，支持批量导入 QBO 系统导出的 Excel 文件，生成批次汇总与 PDF 报表。

  • Excel 导入：直接读取 QBO 导出的 Excel 文件（从第6行起解析），自动建立批次记录
  • 双视图模式：原始数据视图与聚合汇总视图切换，满足不同分析需求
  • 批量选择操作：勾选多条批次，一键标记为已出库或取消标记
  • PDF 导出：通过 jsPDF + AutoTable 生成专业格式批次报表
  • 搜索过滤：按客户、单号、SKU 实时过滤批次列表
  • 云端同步：批次数据支持 Supabase 持久化，多端共享
Excel导入双视图 批量操作PDF报表云端同步

模块 12

库存盘点

基于实物盘点需求，逐货位核对实际库存与系统账面数量，生成盘差报告。

  • 货位盘点：按货架位置逐一盘点，实际数量直接录入对比
  • 差异高亮：系统账面与实盘数量不符时自动标红提示
  • 盘点报告：汇总所有差异，可导出盘差明细
货位盘点差异对比盘差报告

模块 13

货位布局管理

可视化管理仓库货位编码体系，为上架、盘点、下架提供统一的货位数据源。

  • 货位编码维护：新增、编辑、删除仓库货位编码，支持多仓库
  • 布局可视化：以网格方式直观呈现货架位置结构
  • 全局共享：货位数据供上架管理、盘点模块统一调用
货位编码可视布局多仓管理

模块 14

运输资源管理

统一管理司机、车辆、叉车等运输资源档案，为派单分配提供准确的基础数据支撑。

  • 司机管理：司机姓名、联系方式、执照、状态（在职/离职）档案管理
  • 车辆管理：车牌、车型、承重、状态（正常/维修/闲置）全程追踪
  • 叉车管理：叉车编号、型号、工作状态独立台账
  • 车型适配：维护车辆与商品的适配关系，辅助配货决策
  • 车型级联数据：管理品牌/车系/车型三级联动基础数据，供前台使用
司机档案车辆台账 叉车管理车型适配

模块 15

客服库存查询

专为客服岗位设计的库存实时查询工具，支持多仓、多尺码查询及库存预警，提升客服应答效率。

  • 多仓库视图：LS、GT、QC、LLS04 等多个仓库库存分标签展示
  • 智能搜索建议：输入 SKU / 品名时自动提示匹配项，减少拼写错误
  • 库存预警：可配置低库存阈值，零库存与低库存高亮提示，支持 zero / low / all 三档过滤
  • 尺码报表：生成按尺码汇总的库存分布报表，快速识别断货规格
  • 仓库报表：按仓库分类汇总，与客户沟通时精准报出货位库存
  • 同尺码预警：检测同尺码不同 SKU 的替代品，帮助客服推荐替代商品
多仓查询库存预警 尺码报表仓库报表搜索建议

AI 智能推荐

结合库存数据与客户购买历史，通过 AI 分析为客服提供个性化商品推荐，提升成交率。

  • 智能分析：根据客户偏好和当前库存，自动生成推荐商品列表
  • 库存联动：推荐结果实时关联库存数量，确保推荐商品有货
  • 客服辅助：推荐结果可直接在客服通话中引用，提高转化

  • 级联基础数据：维护年份、排量等辅助选车筛选项

模块 19

KPI 数据分析

多维度 KPI 看板，将订单、仓储、运输等业务数据图表化呈现，支撑管理层决策分析。

  • 订单状态分布：饼图展示各状态订单占比（共 29 种状态色）
  • 集装箱统计：在途/到港/卸货/完成状态各柜次数量图表
  • 批次分析：已出库与新建批次对比，评估出库效率
  • 车辆 / 司机状态：正常/维修/闲置车辆占比，在职/离职司机统计
  • 销售统计：销售金额趋势折线图，周期对比分析
  • 收益分析：按类别、区域、时段分析营收来源
订单看板仓储统计 运输状态销售趋势收益分析

模块 20

权限 & 用户管理

基于角色的访问控制体系（RBAC），菜单级细粒度授权，满足仓库、客服、管理等不同岗位的权限需求。

  • 角色管理：自由创建角色（如仓管员、客服、主管），为角色勾选菜单权限
  • 用户管理：系统内部账号的新增、停用、角色分配
  • 网站用户审批：前台商城注册用户的审批上线流程，防止未授权访问
  • 权限树可视化：所有菜单权限以树形结构展示，勾选即授权
  • 动态生效：权限修改后无需重启，下次登录即生效
  • 超级管理员：admin 账号在本地模式下拥有全部权限，云端模式通过角色控制
RBAC菜单级权限 用户审批权限树动态生效
